bioagents

Bio-inspired multi-agent reasoning framework

bioagents is a small experimental framework for goal-directed reasoning with multiple agents that propose ideas, challenge weak ones, and converge on stronger final outputs.

It is designed to stay lightweight: a blackboard, a few rules, heuristic clustering, ranked outputs, and optional local or API-backed language models.

TL;DR

  • multi-agent reasoning
  • goal-directed
  • debate + convergence
  • ranked outputs

How it works

  1. agents propose
  2. agents debate
  3. ideas evolve
  4. clustering
  5. final ranking

Example output

[
  "performance issue from repeated summary recalculation",
  "add a guard before accessing user.profile.email",
  "fix the highest-risk issue first"
]

Quickstart

python -m pip install -e '.[dev]'
bioagents run demos/sample_task.json
bioagents run demos/document_task.json
bioagents run demos/sample_task.json --top-k 2
